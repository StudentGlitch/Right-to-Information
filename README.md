# Right-to-Information
My experiment for vibe coding to make a website hub for financial information

## Getting Started

### Prerequisites

Make sure you have [Git](https://git-scm.com/) installed on your machine.

### Configure Git (Required Before First Commit)

Before you can commit and push code, you need to tell Git who you are. Run the following commands in your terminal, replacing the placeholder values with your own details:

```bash
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
```

**Example:**

```bash
git config --global user.name "StudentGlitch"
git config --global user.email "your.student.id@university.edu"
```

- `user.name` — The name that will appear on your commits.
- `user.email` — The email address associated with your commits (should match your GitHub account email).

> **Tip:** Use the `--global` flag to apply these settings to all repositories on your machine. Omit it to apply the settings only to the current repository.

To verify your configuration, run:

```bash
git config --global user.name
git config --global user.email
```

### Contributing

1. Fork or clone this repository.
2. Configure your Git user details as shown above.
3. Make your changes, then commit and push:

```bash
git add .
git commit -m "Your descriptive commit message"
git push
```
